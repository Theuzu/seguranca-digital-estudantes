# Segurança Digital para Estudantes

Este projeto apresenta um site educativo sobre segurança digital no contexto
universitário. O objetivo é oferecer orientações simples, acessíveis e aplicáveis
ao cotidiano de estudantes que utilizam plataformas acadêmicas, computadores
compartilhados, serviços em nuvem, grupos de comunicação e redes sociais.

O trabalho foi desenvolvido como parte de uma atividade acadêmica de extensão e
busca contribuir para a conscientização sobre hábitos digitais mais seguros,
sem usar linguagem excessivamente técnica ou alarmista.

link do site:

https://seguranca-digital-estudantes.vercel.app/

## Objetivo do Projeto

O site tem como objetivo explicar riscos comuns de segurança digital e apresentar
boas práticas que possam ser adotadas por estudantes no dia a dia. Os conteúdos
foram organizados para facilitar a leitura e incentivar uma postura mais crítica
e responsável no uso da tecnologia.

Entre os temas abordados estão:

- Senhas e verificação em duas etapas
- Cyberbullying e ética digital
- Uso de computadores compartilhados
- Proteção de trabalhos armazenados na nuvem
- Golpes, links falsos e oportunidades suspeitas

## Estrutura do Site

A experiência principal é dividida em três partes:

1. **Introdução**: apresenta o tema e o propósito acadêmico do projeto.
2. **Conteúdo**: reúne os temas de segurança digital com explicações, exemplos e
   checklists práticos.
3. **Conclusão**: reforça os principais cuidados e direciona o visitante para um
   formulário de avaliação.

## Tecnologias Utilizadas

O projeto foi desenvolvido com tecnologias modernas de desenvolvimento web:

- **Next.js**: framework utilizado para construir a aplicação web com estrutura
  baseada em componentes.
- **React**: biblioteca usada para montar as partes visuais e interativas da
  interface.
- **TypeScript**: adiciona tipagem ao código, ajudando a reduzir erros durante o
  desenvolvimento.
- **Tailwind CSS**: utilizado para organizar a estilização da interface de forma
  mais direta e consistente.
- **Motion**: biblioteca usada para animações leves e controladas, sem prejudicar
  a leitura do conteúdo.
- **React Icons**: fornece ícones utilizados na interface.

O conteúdo do site fica centralizado em arquivos TypeScript dentro da pasta
`data/`, o que facilita ajustes futuros no texto sem alterar a estrutura geral
da aplicação.

## Evolução do Projeto

Este projeto ainda pode evoluir conforme o retorno das pessoas que o acessarem. As
respostas do formulário de avaliação serão usadas para identificar pontos de
melhoria, ajustar explicações, revisar exemplos e tornar o conteúdo mais claro
para o público-alvo.

A proposta é que o site continue sendo refinado a partir do feedback real dos
usuários, mantendo o foco em comunicação simples, utilidade prática e
responsabilidade no tratamento dos temas de segurança digital.

## Como Executar Localmente

Para executar o projeto em ambiente de desenvolvimento:

```bash
npm run dev
```

Para verificar a qualidade do código e gerar a versão de produção:

```bash
npm run lint
npm run build
```
