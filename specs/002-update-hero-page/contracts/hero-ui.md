# UI Contract: Hero Page

## Required Visible Text

- Main title: `Seguranca Digital para Estudantes`
- Subtitle: `UNIFBV Wyden - Matheus Maranhao`

## Layout Contract

- Hero stage touches all viewport edges.
- Hero content is centered and visible in the first viewport.
- Reference design influence is visible through title hierarchy, compact label styling if used, cloudy/soft background, subtle retro texture, and centered composition.
- No copied logos, brand names, or unrelated reference imagery.

## Responsive Contract

- Mobile and desktop layouts keep title/subtitle readable.
- No horizontal scrolling.
- Wide viewports keep content centered and constrained enough to avoid stretched text.

## Motion Contract

- Initial animation completes under 2 seconds.
- Motion is CSS-only unless future implementation has explicit approval.
- `prefers-reduced-motion: reduce` minimizes animation while preserving content visibility.
